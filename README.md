# 🔐 Caesar Cipher in Python

This is a simple Python program that implements the **Caesar Cipher** encryption technique. It allows users to encode (encrypt) and decode (decrypt) messages using a shift value.

---

## 🚀 Features

* Encode messages (encryption)
* Decode messages (decryption)
* Handles non-alphabet characters (numbers, symbols, spaces)
* Supports continuous use (loop until user exits)
* Works with any shift value (automatically wraps around alphabet)

---

## 🛠️ How It Works

The Caesar Cipher shifts each letter in the message by a fixed number of positions in the alphabet.

Example:

* Input: `hello`, Shift: `3`
* Output: `khoor`

---

## ▶️ How to Run

1. Make sure Python is installed
2. Clone this repository:

   ```bash
   git clone https://github.com/Krish-Ahirrao/Caesar-Cipher.git
   ```
3. Navigate to the project folder:

   ```bash
   cd Caesar-Cipher
   ```
4. Run the program:

   ```bash
   python main.py
   ```
